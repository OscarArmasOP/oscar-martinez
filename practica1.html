HTML Y CSS

En esta practica manejamos estilos para nuestra página web, estilos con CSS, aprendiendo las diferentes cosas que conlleva CSS. A mi parecer CSS es un lenguaje para el diseño que tiene su grado de dificultad, sobre todo al momento de acomodar los contenidos que manejamos en nuestro HTML, y digo grado de dificultad por que existen demasiadas propiedades con muchas variantes y dominarlo al 100% es algo que lleva mucha práctica y tiempo, sin embargo es muy flexible para el aprendizaje a démas de ser escencial para una página web.
